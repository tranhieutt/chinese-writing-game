📄

App Shell (HTML)

Định nghĩa DOM contract, load stylesheet, data file và game logic theo thứ tự cố định.

⚙️

Game Controller (JS)

Owns toàn bộ state machine, xử lý HanziWriter events, scoring, XP, confetti và audio.

✍️

HanziWriter CDN

Thư viện vẽ nét chữ Hán. Load từ jsDelivr → unpkg theo thứ tự fallback để đảm bảo availability.

💾

localStorage

Lưu trữ XP, streak, ngày học và ký tự đã hoàn thành qua key prefix hz_*.

Loading
⏳ Loading
HanziWriter đang được tải từ CDN. Tất cả controls bị disable.
Idle
○ Idle
Trạng thái chờ. User có thể chọn nhóm, ký tự hoặc bắt đầu preview/quiz.
Preview
▶ Preview
HanziWriter đang animate nét chữ. Navigation bị lock đến khi animation kết thúc.
Quiz
✏️ Quiz
User đang vẽ nét. Mỗi nét đúng +10 XP, nét sai tăng attempt counter.
Done
🎉 Done
Hoàn thành ký tự. Confetti, completion XP, streak update và overlay hiển thị.
Error
❌ Error
CDN failure. Status bubble hiển thị lỗi, game controls bị vô hiệu hóa hoàn toàn.

# Từ Đến Thông điệp / Action Ghi chú
01 APP → VOCAB read groups and chars Startup: đọc 7 nhóm từ vựng HSK1
02 APP → DOM render group tabs, char selector buildGroupTabs() + buildCharSelector()
03 User → DOM select character Click .char-btn
04 DOM → APP selectChar(index) Trigger qua onclick attribute
05 APP → HW create writer(currentChar) initWriter() — mount vào #hanzi-container
06 HW → APP charDataLoader(char) HW yêu cầu stroke data
07 APP → CDN fetch char JSON jsDelivr hoặc unpkg fallback
08 CDN ↩ APP strokes, medians, radStrokes JSON chứa đủ thông tin vẽ nét
09 User → DOM draw stroke Pointer/touch event trên canvas
10 HW → APP onCorrectStroke / onMistake Callback từ HanziWriter quiz API
11 APP → DOM update progress, score, mascot Stroke dots, progress bar, mode badge
12 APP → LS persist XP, streak, completed chars Keys: hz_xp, hz_streak, hz_completed_chars
